Api

Name Type Info
onChange function(newVal, oldVal) The function to call when a new tag added or removed. newVal contains the current value of the tags, oldVal contains the previous value of the tags.
addKeys Array Array of keyCodes, defaults to Enter (13)
removeKeys Array Array of keyCodes, defaults to Backspace and Delete (46, 8)
upKey Number KeyCode that navigates up in the suggestions dropdown, defaults to Up arrow (38)
downKey Number KeyCode that navigates down in the suggestions dropdown, defaults to Down arrow (40)
suggestions Array / function(searchString) Suggestions to show in the dropdown menu. If it's a function it must return a promise with a then() method. See the movies example for sample implementation.
tags Array Pre-selected tags.
